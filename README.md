# Translation Style Guide Project
## Проект разработки стандартов перевода книг по соматической психологии

---

## Структура репозитория

```
/
├── source/
│   ├── en/                 ← оригиналы на английском (DOCX, PDF, TXT)
│   └── ru/                 ← существующие переводы (эталон для анализа)
├── analysis/
│   ├── 00_WORKFLOW_ПЕРЕВОДА.md    ← методология перевода
│   ├── 01_ОГЛАВЛЕНИЕ.md           ← структура книги
│   ├── glossary.md                ← глоссарий терминов (TODO)
│   ├── style_guide.md             ← стилистические правила (TODO)
│   └── parallel_corpus.md         ← параллельные тексты (TODO)
├── translations/           ← новые переводы
├── .gitattributes          ← настройки Git LFS для PDF
└── README.md
```

---

## Как добавить PDF файлы

Git LFS уже настроен для PDF. Просто:

```bash
# 1. Положите PDF файлы в нужную папку:
cp ваш_файл.pdf source/en/   # для английского
cp ваш_файл.pdf source/ru/   # для русского

# 2. Добавьте и закоммитьте:
git add source/
git commit -m "Добавлены исходные PDF файлы"
git push
```
